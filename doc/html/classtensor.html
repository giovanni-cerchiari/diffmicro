Detailed Description

template<typename FLOAT>
class tensor< FLOAT >

This class implements a tensor of generic type for memory managment. Particularly useful for high-performance computing or to operate directly inside the memory is the possibility to use high-level function for allocation and delete, but at the same time to get the pointer to memory when it is needed.
