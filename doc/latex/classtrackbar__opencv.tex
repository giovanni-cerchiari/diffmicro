\doxysubsection{Detailed Description}
A trackbar is a user interface object that can have integer value between 0 and value\mbox{[}0\mbox{]} The values assumed by the trackbar are store in value vector. Each value has its handle to allow for separate manipulation. See \mbox{\hyperlink{classcontrol__opencv}{control\+\_\+opencv}} for further details.
