\doxysubsection{Detailed Description}
This functions are written for diffmicro.\+exe application.

This file contains the functions that execute the main algorithm on cpu and gpu. Execution on the different hardware is implemented with minimal modifications on the main algorithm by using function pointers. Function pointers selection is performed in

This structure and the function void \mbox{\hyperlink{correlation_8h_a1450ba6698042b271c113e15436b1bac}{calc\+\_\+sizes(\+INDEX dimy, INDEX dimx, INDEX n\+\_\+tot, INDEX size\+\_\+of\+\_\+element, sizes \&s)}} allow to store in a single place the values concerning the memory managment of some allocated memory. To understand the working principle we will make the example an image. To store a series of images the following data are saved
\begin{DoxyItemize}
\item dimension in pixels dimx, dimy and dim = dimx$\ast$dimy
\item numerosity -\/$>$ the stored number of images
\item memory\+\_\+one -\/$>$ memory size in byte of one image
\item memory\+\_\+tot -\/$>$ memory size in byte for the total number of images
\item cexe -\/$>$ CUDA \#blocks and \#threads necessary to browse all the pixel of a single image as a linear vector 
\end{DoxyItemize}
