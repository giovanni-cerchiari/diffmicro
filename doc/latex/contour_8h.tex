This set of funtions are inteded to find contour courve in a contour-\/plot like behaviour

This function might be used togheter with equipoints. It makes an estimation about the deviation of the points that could constitue a line of equal-\/value points. In practice it is worthless to ask for the same value, but better to similar value. This function attempts a definition of similarity. The idea is that in a courour plot it is possible to draw a line. Therefore considering a matrix around a coutour point the deviation is estimated as max and min. The deviation to be considered is the one that allows to pick up approximately the points that would allow a line to go through the matrix.
