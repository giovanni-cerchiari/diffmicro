\doxysubsection{Detailed Description}
This is a special trackbar that changes the dynamic range of an image. The dynamic range have a maximum and a minimum value. With this trackbar it is possible to modify maximum, minimum and average value within an interval See \mbox{\hyperlink{classtrackbar__opencv}{trackbar\+\_\+opencv}} and \mbox{\hyperlink{classcontrol__opencv}{control\+\_\+opencv}} for further details
