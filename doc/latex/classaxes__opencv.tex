\doxysubsection{Detailed Description}
This overlay implements the 2-\/D axes on the \mbox{\hyperlink{classwindow__display__opencv}{window\+\_\+display\+\_\+opencv}}. It is used with \mbox{\hyperlink{classscatter__opencv}{scatter\+\_\+opencv}} and \mbox{\hyperlink{classbarplot__opencv}{barplot\+\_\+opencv}}. Plots are implemented as overalay to facilitate plotting over images.
