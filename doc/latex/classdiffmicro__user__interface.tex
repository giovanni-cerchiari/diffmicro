\doxysubsection{Detailed Description}
This class is written for diffmicro application. This class stores all the user preferences about the execution of the program.

This file contains the functions and classes used to communicate with the program by the user. The user interface is meant to be a text file with a specific structure that can be reconstructed via the provided graphical user interface.
