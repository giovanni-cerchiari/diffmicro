\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename INT$>$\newline
class circular\+\_\+index$<$ INT $>$}

This class implements modular arithmetics on index values of a specified index in a defined interval \mbox{[}min, max\mbox{]} . First and last listed values are included.

Only incremental operation has been implemented like operators +, -\/, ++, -- .

A circular index must incremented or decremented by its methods, without making operations directly on .i attribute, that has been left public to speed up the variable reading!!!

Pay attention that no error is returned if you try to assign to a \mbox{\hyperlink{classcircular__index}{circular\+\_\+index}} a value outside \mbox{[}min, max\mbox{]} interval, because it is forced, using periodicity criteria, into the \mbox{[}min, max\mbox{]} interval.

\mbox{\hyperlink{classcircular__index}{circular\+\_\+index}} is unsigned safe (INT = unsigned int).

Initialization (init(INT \+\_\+max, INT \+\_\+min = 0, INT val = 0)) is not safe, choose min$<$=val$<$=max and if you assign INT = unsigned and do not try to initialize min $<$ 0.

Comparison operators has been implemented, but use $<$, $>$, $<$=, $>$= with extream care because theoretically a \mbox{\hyperlink{classcircular__index}{circular\+\_\+index}} cannot be grater than another due to the periodicity condition.
